slug: oi-tam-u-lisi-kalyna
title: Ой там у лісі калина
category: rusalni-ta-petrivchani-pisni
---
```
Ой там у лісі калина,
Під калиною дівчина.
Там косу русу чесала,
Там біле личко вмивала.
```

```
Ой ти, личенько біленьке,
Кому ж ти будеш миленьке?
Чи ратайку, — дай Боже!
Чи лінтяйку, — крий Боже!
```

```
Бо ратайко працює,
А лінтяйко балує.
Бо ратайко в полі є,
А лінтяйко в корчмі п’є.
```
