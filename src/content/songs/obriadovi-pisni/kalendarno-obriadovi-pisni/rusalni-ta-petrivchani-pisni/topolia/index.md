Обмаєну та прибрану намистом і стрічками дівчину водили на Полтавщині по селу, співаючи:

```
Стояла тополя 
Край чистого поля.
```

*Приспів:*

```
Стій, тополенько, 
Стій, не розвивайсь, 
Буйному вітроньку 
Не піддавайсь. [1]
```

```
На тій тополі 
Чотири сокілки.
 Перва сокілка —
Горпинка-дівка,
 Друга сокілка —
 Галочка-дівка, 
Третя сокілка —
 Вірочка-дівка, 
Четверта сокілка —
 Катричка-дівка.
```

## Примітки

[1] Повторюється після кожного рядка
