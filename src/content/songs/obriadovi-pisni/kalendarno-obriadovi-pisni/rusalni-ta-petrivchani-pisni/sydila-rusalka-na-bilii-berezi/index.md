```
Сиділа русалка на білій березі.
Просила русалка в жіночок намітки:
— Жіночки сестрички,
Дайте мені намітки,
Хоч не тоненької,
Аби біленької.
```

```
Сиділа русалка на білій березі,
Просила русалка в дівочок сорочки:
— Дівочки-сестрички,
Дайте мені сорочки,
Хоч не біленької,
Аби тоненької.
```
