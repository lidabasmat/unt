title: Ой рано, рано

---
```
Ой рано, рано
Кури запіли.
```

```
Приспів:
Гей, калино,
Гей ти, червона.
```

```
А ще раніше
Молодець устав
```

```
Молодець устав,
Дзвунком подзвонив.
```

```
Дзвунком подзвонив,
Браття побудив.
```

```
— Уставайте, браття,
Коня усідлати.
```

```
Та поїдемо
У чисте поле.
```

```
У чисте поле
На полюваннячко.
```

```
То сполюємо
Тура й оленя.
```

```
То сполюємо
Й роги здіймемо.
```

```
Нам тії роги
Дуже способни.
```

```
Ми тії роги
В стіну ввіб'ємо.
```

```
Свому батейку
Вішать шабельку.
```

```
Своїй сестричці
Вішать платочки.
```
