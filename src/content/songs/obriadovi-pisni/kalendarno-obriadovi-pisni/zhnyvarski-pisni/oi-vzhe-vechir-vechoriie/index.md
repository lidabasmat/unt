title: Ой вже вечір, вечоріє
category: zhnyvarski-pisni
---
```
Ой вже вечір вечоріє,
Сонце над вербами.
Пускай, пане отамане,
Додому з серпами!
```

```
Ой вже вечір вечоріє,
Уже сонце низько.
Пускай, пане отамане,
Додому неблизько!
```

```
Скажи мені, отамане,
Скажи мені, пане,
А хто вийде жито жати,
Коли нас не стане?
```
