---
slug: oi-zhny-divcha-pshenychenku
title: Ой жни, дівча, пшениченьку
author: Український фольклор
category: zhnyvarski-pisni
---
```
— Ой, жни, дівча, пшениченьку
Та й не думай собі,
Буде тота пшениченька
На коровай тобі!
```

```
А як будеш, дівчинонько,
Думати, думати,
Буде твоя пшениченька
В полі зимувати.
```

```
Ти, дівчино чорнявая,
Ти, дівчино біла,
Чи ти в полі не робила,
Що ж не обгоріла?
```

```
— А я в полі, легінику,
Жала і в'язала,
Але своє біле личко
Хустков закривала.
```

```
— Якби ж була, дівчинонько,
Того не сказала,
Була-бись ся, моя любко,
Сих м'ясниць віддала.
```

```
— Ой, я в полі та робила
І робити буду.
Але-м роду біленького
Та й біленька буду.
```
