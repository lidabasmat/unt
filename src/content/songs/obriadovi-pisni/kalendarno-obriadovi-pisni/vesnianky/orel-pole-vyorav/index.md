```
Орел поле виорав, (2)
 Пшеницю засіяв, (2)
 Крилечками зволочив, (2) 
Дрібний дощик примочив.  (2)
На пшеницю урожай, (2)
 А дівонькам коровай (2)
 Молодицям на крохмаль.  (2)
```

```
Орел поле виорав (2)
 Ще й вівсиком засіяв, (2)
 Крилечками зволочив, (2)
 Дрібний дощик примочив.  (2)
На вівсисько урожай, (2)
 А парубкам коровай, (2)
 Старим бабам на крохмаль. (2)
```
