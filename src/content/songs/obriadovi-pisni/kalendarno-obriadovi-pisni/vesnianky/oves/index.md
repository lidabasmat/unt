```
Ой посієм овес
Та до зерна увесь.
```

*Приспів:*

```
До зерна, до зерна,
До зерна увесь. [1]
```

```
Як уродить же овес.
Як покосимо овес.
Як пов’яжемо овес.
Як покладемо овес.
Як повозимо овес.
Помолотимо овес.
Як повіємо овес.
Повбираємо овес.
Погодуємо овес
Та до зерна увесь.
```

## Примітки

[1] Повторюється за кожним рядком.
