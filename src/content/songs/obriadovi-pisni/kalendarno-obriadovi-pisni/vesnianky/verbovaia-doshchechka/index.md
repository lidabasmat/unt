```
Вербовая дощечка,
По ній ходить Настечка.
```

```
На всі боки леліє,
Відкись милий приїде.
```

```
Він Настунці привезе,
Жовтенькії чобітки.
```

```
Жовтенькії чобітки,
А з позлітки підківки.
```

```
Іх зозулька підкує —
І Настунька обує.
```
