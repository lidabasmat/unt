---
slug: syroty-za-matiriu
title: Сироти за матір’ю
category: holosinnia
---
```
Мамочка моя пчілочка,
мамочка моя ластівочка,
словечко моє вірнеє,
зезулечко моя лугова —
нащо ти мене кинула?
```

```
Нема кому мені правди сказати,
ні на кого мені діток бросити маленьких,
нема до кого мені прийти,
ні з ким мені поговорити.
```

```
Откуди ти до мене прилетиш?
Із якої сторононьки мені тебе виглядати?
Сонечко моє яснеє,
ручечки мої пишнії,
ніжечки мої ходулечки!
Находились твої ніжечки,
наробились твої ручечки!
```

```
Очечки твої карії; надивились твої очечки.
Птичка моя одлетная!
Одлетіла ти од мене, моя мамочко,
не буду вже я тебе бачити,
личенько твоє білеє.
Чому ти не встанеш
да не змовиш словечка свого правдивого?
```
