---
slug: za-ditmy-malymy
title: За дітьми малими
category: holosinnia
---
```
Дитятко моє, любе та миле!
Куди ж ти убираєся?..
Відкіля тебе визирати, відкіля тебе виглядати?
```

```
Чи з церкви ітимеш,
Чи з дороги їхатимеш?..
Чи тебе соловейки одщебетали,
Чи тебе зозуленьки одкували?
```

```
Як твоя доріжечка смутна та невесела!
Яка твоя хаточка темна та невидная,
Що ні дверичок, ні оконечка, ні праведного сонечка…
Як яблочко од яблоньки одкотилося,
Так ти од мене, моє дитяточко, одрознилося.
```

```
І відкіля ти ітимеш,
І яку вісточку нестимеш, моя донечко?
Моя донечко, моя ягідочко,
На що ж ти мене кидаєш?
```

```
Хто ж буде тебе доглядати тамечки?
І коли ж ти до мене у гості прийдеш?
Коли ж тебе дожидать, коли ж тебе сподівать?
```

```
Примай мене до себе,
Пригортай мене до себе!
```

```
Обирай мені місцечко коло себе!
Де тобі місцечко найшлося гарне,
Що ти мене покинуло?
```

```
І серце моє одірвалося!
Дитятко моє і голубятко моє!
Коли я тебе побачу?
```

```
Чи ти зозулею прилетиш, чи соловейком?
Чи ти нам перекажеш, чи письмо пришлеш,
Щоб я почула, де ти будеш!
```

```
Моє соколятко, моє голубятко!
```
