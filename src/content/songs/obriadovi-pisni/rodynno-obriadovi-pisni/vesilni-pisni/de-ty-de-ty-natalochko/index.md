---
slug: de-ty-de-ty-natalochko
title: Де ти, де ти, Наталочко
author: Український фольклор
category: vesilni-pisni
---
```
— Де ти, де ти, Наталочко,
Де ходила,
Що ти собі Іваночка
Полюбила?
```

```
— Та ходила ж я, подруженьки,
У сад по воду,
Найшла собі товариша
Чесного роду.
```

```
Та ходила ж я, подруженьки,
По крутій горі,
Знайшла собі Іваночка,
Милого собі.
```

```
— На віщо ти, Наталочко,
Уповала,
Що ти собі Іваночка
Сподобала?
```

```
Та чи на його кониченьки
Воронії,
Чи на його жупани
Голубії?
```

```
Та чи на його степи-поля,
Чи на гроші,
Чи на його біле личко,
Що хороше?
```

```
Та єсть у нього кониченьки,
Та то ж не його,
Єсть у нього, Наталочко,
Братічок його.
```

```
Та єсть у нього степи-поля,
Та то ж не його,
Єсть у нього, Наталочко,
Батечко його.
```
