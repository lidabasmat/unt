```
Одна гора високая,
А друга низька...
Одна мила далекая,
А друга близька.
```

```
У цієї близенької
Воли та корови,
А в тієї далекої
Та чорнії брови.
```

```
У цієї близенької
Воли поздихають,
А в тієї далекої
Брівки не злиняють.
```

```
У цієї близенької
Рушник на кілочку,
А в тієї далекої
Брови на шнурочку.
```

```
Ой, я цюю близенькую
Людям подарую,
А до тої далекої
Пішки помандрую!
```
