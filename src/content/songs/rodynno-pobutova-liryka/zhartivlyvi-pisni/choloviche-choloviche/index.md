```
Чоловіче, чоловіче,
Ти дай мені раду:
Я з твоїми коноплями
Навіки пропаду.
```

```
В людей було по сім кіп
Та й вже всі по стерті,
А у мене одна горстка —
Ворота підперті.
```

```
В людей було ще й побільш,
Та й тії потруться,
А у мене одна горстка, —
Горобці несуться.
```

```
Як узялась я на три дні
Та й горстку постерла,
Бідна ж моя головонько,
Аж трохи я не вмерла!
```

```
Не сама ж я пряла:
Кума помагала,
А я кумі — миску муки
І два куски сала.
```

```
Не сама й мотала:
Мала помічницю,
Дала їй миску муки
Та ще й паляницю.
```

```
Не сама й завивала:
Наймала Данила,
Дала йому два золоті
Та ще й завинила.
```

## Примітки

Останні два рядки кожного стовпчика повторюються двічі
