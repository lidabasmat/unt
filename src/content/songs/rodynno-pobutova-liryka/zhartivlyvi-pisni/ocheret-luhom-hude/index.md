title: Очерет лугом гуде
author: Український фольклор
category: zhartivlyvi-pisni
---
```
Очерет лугом гуде,
Сова річкою бреде.
Підіймає сова крильця,
Виглядає чорнобривця.
```

```
Чорнобривець іде,
На добридень дає:
— Добридень тобі,
Бо ти жінка моя.
```

```
Бо ти жінко моя,
Бо ти любка моя,
Бо ти любиш мене,
Бо ти підеш за мене.
```

```
— Я й не жінка твоя,
Я й не любка твоя,
Я й не люблю тебе,
Я й не піду за тебе.
```

```
Бо ти тонкий та високий,
Ще й поганий, косоокий,
Ще й без верхньої губи,
Як цілуєш, видно зуби.
```

```
Такий стан, як у баби,
Такі очі, як у жаби,
Такі вуса, як у рака,
А сам рудий, як собака.
```

```
Такі вуса, як у рака,
А сам рудий, як собака,
Ще й на ногу налягаєш,
Бо на плечах горба маєш.
```

```
— Чи я, мамцю, не доріс,
Чи я, мамцю, переріс?
Чи не врублена хата,
Що не люблять дівчата?
```

```
— Ні, ти, синку, і доріс,
І ти, синку, переріс,
І дорублена хата,
Лиш не люблять дівчата.
```

```
Піди, синку, до криниці
Та й напийся водиці,
Так любитимуть дівчата,
Любитимуть молодиці.
```

```
— А я йду і бреду
По коліна в лободу,
А я піду до дівчини,
Що хороша на виду.
```

## Примітки

Останні два рядки кожного стовпця повторюються двічі.
