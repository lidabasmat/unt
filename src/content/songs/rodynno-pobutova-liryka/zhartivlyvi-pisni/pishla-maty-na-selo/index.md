```
Пішла мати на село
Гречну муку добувати,
Гречаники учиняти,
Своїх діток годувати.
```

*Приспів:*

```
Гоп, мої гречаники,
Гоп, мої білі,
Чогось мої гречаники
На скорині сіли.
Гоп, мої гречаники,
Гоп, мої смачні,
Чогось мої гречаники
Не вдалися вдачні.
```

```
Молов батько, не віявши,
Пекла мати, не сіявши,
З помийниці воду брала,
Гречаники учиняла.
```

```
Іди, старий, до криниці,
Іди за водою,
А ми зварим обідати
Удвох із кумою.
```

```
Зараз кашу без пшона,
Без солі зварила,
І без хліба татусенька
Їсти посадила.
```

```
Сидить тато в кінці стола,
Опустивши вуха, —
Усякому так буває,
А хто жінок слуха.
```
