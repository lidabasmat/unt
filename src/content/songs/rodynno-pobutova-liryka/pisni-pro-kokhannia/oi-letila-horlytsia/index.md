```
Ой, летіла горлиця через сад,
Через сад, гей!
Розпустила пір'ячко на ввесь сад,
Гей, на весь сад!
```

```
А хто ж теє пір'ячко ізбере,
Ізбере, гей!
Отой мене, молодую, забере,
Гей, забере!
```

```
На курочці пір’ячко в один ряд,
В один ряд, гей!
Любімося, серденько, в один лад,
Гей, в один лад!
```
