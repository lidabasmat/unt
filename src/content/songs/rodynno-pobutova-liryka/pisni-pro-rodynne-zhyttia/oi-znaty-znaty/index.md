```
Ой знати, знати,
Хто кого любить:
Сяде близесенько
Ще й приголубить.
```

```
Ой знати, знати,
В кого є дочки:
Втоптані стежечки
Через садoчки.
```

```
Ой знати, знати,
Хто нежонатий:
Білеє личенько,
Як в паненяти.
```

```
Ой знати, знати,
Хто оженився:
Скорчився, зморщився
Та й зажурився.
```

## Примітки

Останні два рядки кожного стовпчика повторюються двічі.
