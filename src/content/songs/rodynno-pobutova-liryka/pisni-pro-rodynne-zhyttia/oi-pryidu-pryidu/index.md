---
slug: oi-pryidu-pryidu
title: Ой прийду, прийду
author: Український фольклор
category: pisni-pro-rodynne-zhyttia
---
```
Ой пряду, пряду,
Спатоньки хочу...
Ой, склоню я голівоньку
На білую постілоньку,
Може, я засну.
```

```
Аж свекруха йде,
Як змія, гуде:
— Сонливая-дрімливая,
До роботи лінивая
Невістко моя!
```

```
Аж свекорко йде,
Як вітер, гуде:
— Сонливая-дрімливая,
До роботи лінивая
Невістко моя!
```

```
Ой пряду, пряду,
Спатоньки хочу...
Ой, склоню я голівоньку
На білую постілоньку,
Може, я засну.
```

```
А мій милий йде,
Як голуб, гуде:
— Ой спи, мила-хорошая,
Пішла заміж молодая,
Не виспалася!
```
